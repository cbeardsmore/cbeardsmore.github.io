Policy. Effective 9 May 2020.

Fined stores the minimum required amount of data needed to provide the service required by its users. Our code is open source and can be viewed on Github.

Stored Data

We store the following data as used by the Application. Team and Channel Ids to connect your channels fines to your workspace. Username of the user who used the /fine command. Details of the /fine text as entered. No further data is stored. Fined's scope is limited to the "command" OAuth scope. It cannot read information about workspace users and does not have access to workspace messages. We also use Google Analytics on this website to track user views and where they are refered from.

Data Retention

Data is stored only as long as required. Fines are deleted when marked as paid and no history is saved. We use AWS and data is stored securely within DynamoDB.

Privacy

Fined will never use tracking data or sell tracking data to any 3rd party. We will also never allows Ads or Ads tracking. Data stored will also never be shared publicly.

Changes to this policy

This policy may change and be updated over time. If you have any questions about how we handle data, please contact us. You are advised to review this Privacy Policy periodically for any changes. Changes to this Privacy Policy are effective when they are posted on this page.

Bugs and feature requests

Fined is an open-source project. Report bugs, submit feature requests and contribute to the project on GitHub.
